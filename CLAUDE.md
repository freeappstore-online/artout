# ArtOut

Location-based street art photo sharing PWA on FreeAppStore.

- Subdomain: `artout.freeappstore.online`
- Dev: `pnpm install && pnpm dev`
- Build: `pnpm build`
- Deploy: `git push origin main` (auto-deploys via Cloudflare Pages)

## Architecture

- **Data:** FAS Collections `posts` collection (public read, auth write)
- **Auth:** FAS SDK (`fas.auth.signIn('google')`)
- **Favorites:** FAS KV (per-user, key: `favorites`, value: `string[]`)
- **Images:** Cloudinary (direct client upload, cloud: `lkzycqsuf`, preset: `cttcla3s`)
- **Map:** Google Maps JS API via `@vis.gl/react-google-maps`
- **Location tree:** Derived client-side from posts' `locationPath` field (no separate collection)
- **Gallery:** `yet-another-react-lightbox`

## Data model

```typescript
interface ArtPost {
  id: string           // auto-generated by FAS
  imageId: string      // Cloudinary public_id
  imageUrl: string     // Cloudinary full-res URL
  thumbUrl: string     // Cloudinary thumbnail URL
  title?: string
  description?: string
  lat: number
  lon: number
  locationPath: string // "Australia > Melbourne > Fitzroy"
  locationName: string // "Fitzroy"
}
```

## Env vars

- `VITE_GOOGLE_MAPS_KEY` -- Google Maps JavaScript API key
